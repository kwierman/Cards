Detailed Description

Provides logging functionality.

Author
Kevin Wierman

Typical usage is to:

  • Set the logfile:
    • CardLog.set_logger_file("yourlogfile.log")
  • Log a header:
    • CardLog.log(CardLog.Severity.Header, "Your Header")
  • Log a line:
    • CardLog.log_line()
  • Log a normal message:
    • CardLog.log(CardLog.Severity.Normal, "Your Message")
  • Etc...
