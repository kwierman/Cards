Detailed Description

Provides logging functionality.

Author
Kevin Wierman

Typical usage is to:

  • Set the logfile:
    • Log.set_logger_file("yourlogfile.log")
  • Log a header:
    • Log.log(Log.Severity.Header, "Your Header")
  • Log a line:
    • Log.log_line()
  • Log a normal message:
    • Log.log(Log.Severity.Normal, "Your Message")
  • Etc...
