Cards.Card.Ranks¶

Defines the numerical values for card suits. The Ranking is as follows:

  1. Non-suit type for null-cards (value=Suits.no_suit)
  2. Hearts (value=Suits.hearts)
  3. Spades (value=Suits.spades)
  4. Diamonds (value=Suits.diamonds)
  5. Clubs (value=Suits.clubs)

Log Module¶

@package Log Provides logging functionality

@author Kevin Wierman

Typical usage is to:
  • Set the logfile: - Log.set_logger_file(“yourlogfile.log”)
  • Log a header: - Log.log(Log.Severity.Header, “Your Header”)
  • Log a line: - Log.log_line()
  • Log a normal message: - Log.log(Log.Severity.Normal, “Your Message”)
  • Etc...
